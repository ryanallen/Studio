# Skills

Reusable workflows by trigger phrase or `/skill-name`. One kebab-case folder per skill with `SKILL.md`. **Verify task** first every turn: `npm run checklist -- "<summary>"` ([verify-task](.claude/skills/verify-task/SKILL.md)). Flow-invoked skills use `disable-model-invocation: true`; see [README Frontmatter](README.md#-frontmatter-agents-and-skills).
